\section{Evaluasi}

Untuk menguji pemahaman Anda terhadap materi Bab 1, silakan jawab pertanyaan-pertanyaan berikut:

\begin{enumerate}
    \item Jelaskan perbedaan mendasar antara kompilator dan interpreter dalam hal eksekusi program dan output yang dihasilkan!
    
    \item Sebutkan dan jelaskan secara singkat tiga fase utama dalam Front-End kompilator!
    
    \item Apa fungsi utama dari Analisis Semantik (Semantic Analysis)? Berikan satu contoh kesalahan yang dapat dideteksi oleh fase ini!
    
    \item Mengapa Generasi Kode Intermediate (Intermediate Code Generation / IR) penting dalam arsitektur kompilator modern? Jelaskan hubungannya dengan portabilitas!
    
    \item Apa tujuan utama dari Optimasi Kode (Code Optimization)? Mengapa optimasi perlu memperhatikan waktu kompilasi?
\end{enumerate}
