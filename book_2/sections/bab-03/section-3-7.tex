\section{Handling Komentar dan Kesalahan}

\subsection{Whitespace dan Komentar}
Kompilator biasanya mengabaikan spasi, tab, dan baris baru. Komentar (\texttt{//} atau \texttt{/*...*/}) juga harus diabaikan tanpa menghasilkan token ke parser.

\begin{lstlisting}[language=C++]
void Lexer::skipLineComment() {
    get(); get(); // skip //
    while (peek() != '\n' && peek() != '\0') get();
}
\end{lstlisting}

\subsection{Penanganan Kesalahan Leksikal}
Jika lexer menemui karakter yang tidak dikenal atau string yang tidak tertutup hingga akhir file (\textit{unclosed string}), maka harus dihasilkan informasi kesalahan yang informatif.
